================
PASSO 2: EXEMPLO
================

O Water Framework já vem com uma aplicação de exemplo, e para utilizá-la
é necessário criar o banco e a tabela conforme abaixo:

Usando MYSQL:
-------------

CREATE DATABASE water;
USE water;
CREATE TABLE users (
	id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
	name VARCHAR(50) NOT NULL,
	email VARCHAR(50) NOT NULL UNIQUE,
	password VARCHAR(100)
);

OBS: Você pode criá-lo em outro SGBD de sua preferência e atualizar o
driver de conexão com o banco em "app/config/config.php".
